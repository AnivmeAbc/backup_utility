# Утилита резервного копирования ключевых папок

Простое desktop-приложение на Python с графическим интерфейсом (Tkinter) для создания резервных копий выбранных папок.

## Функциональность
- Выбор исходной папки через диалог
- Автоматическое создание папки бэкапа с меткой даты и времени
- Полное копирование структуры папок и файлов
- Ведение лога выполнения (backup_log.txt)
- Индикатор прогресса и статус выполнения
- Обработка ошибок с уведомлениями

## Требования
- Python 3.6+
- Стандартная библиотека (tkinter, shutil, os и др.)

## Запуск
```bash
python main.py
